### 3つの柱 各コピー

**1. 大会運営**
```
タイトル：舞台を作り続ける。

大会は、プレイヤーに主人公になる機会を渡す行為だ。
テンプレートを整備し、継続開催の仕組みを作った。
一回きりで終わらない。それが俺たちの約束だ。
```

**2. アンバサダー制度**
```
タイトル：コミュニティの顔に、なれ。

アンバサダーは、担当ゲームの場を作る人間だ。
新しいメンバーを迎え入れ、参加者同士をつなぎ、コミュニティに温度を作る。
「管理」じゃない。「場作り」だ。
```

**3. 配信者部門 Regal Rise**
```
タイトル：フォロワーは関係ない。本気があるかだ。

Regal Rise 配信者部門は、発信力をゼロから育てる場所だ。
フォロワー数は見ない。チャンネル登録数も見ない。
本気でゲームを発信したいなら、ここで始めろ。
```

## PAGE 3 — ambassador.html（アンバサダー募集専用ページ）

---

### ページ ヒーロー大見出し（3案）

```
案A（推し）：コミュニティの顔に、なれ。
案B：場を作る側に、立て。
案C：お前が、この場所を変える。
```

### 推し案：案A
brief.mdに「コミュニティの顔に、なれ。」と明記されている。命令形の断定一文で、読んだ瞬間に自分事として受け取れる。短さで殴る情熱モードの核心。

---

### サブキャッチ

```
スキル不問。でも本気の人間だけが、この場所を変えられる。
```

---

### 「アンバサダーとは」説明

```
アンバサダーは、コミュステの各ゲームを担当する"場作りの主役"だ。
大会を動かすのは運営チームだが、コミュニティに温度を作るのはアンバサダーの仕事だ。
新しいプレイヤーが「ここは居心地がいい」と感じる場所を、お前の手で作れ。
```

---

### 役割説明 各コピー

**場作り**
```
タイトル：舞台の空気を、作れ。

担当ゲームのチャンネルを動かし、コミュニティに熱量を生む。
チャットひとつ、声かけひとつが、場の温度を変える。
それがアンバサダーの仕事の核心だ。
```

**交流促進**
```
タイトル：つなぐ。それが力だ。

初めて来た人間が迷子にならないように。
ベテランと新参者が自然に話せるように。
人と人をつなぐことが、コミュニティを強くする。
```

**新規歓迎**
```
タイトル：最初の一声が、全部を決める。

新しいメンバーへの最初の声かけが、その人の居場所を決める。
「来てよかった」と思わせるのは、制度じゃなくて人間だ。
お前の一声が、誰かの始まりになる。
```

---

### 特典コピー 各セクション

**専用ロール**
```
コミュステ内で「アンバサダー」ロールが付与される。
それはただのバッジじゃない。この場所での影響力だ。
```

**企画権限**
```
大会・イベントの企画を、自分の手で動かせる。
「やってみたい」を「やった」に変える権限を持つ。
```

**RegalRise参加機会**
```
実績が積み上がれば、RegalRiseチームへの参加機会が生まれる。
コミュニティから、チームへ。その道は本物だ。
```

---

```
- ゲームが好きで、その熱量を誰かと分かち合いたいと思っている人間
- 「もっとこのコミュニティをよくしたい」という気持ちがある人間
- 自分から声をかけることを、面倒だと思わない人間
- スキルより、場の空気を大事にできる人間
```
