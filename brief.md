## デザイン DNA
movie.htmlの世界観を継承・拡張する。

| 要素 | 指定 |
|---|---|
| 背景 | #0A0A0A（純粋な黒） |
| アクセント | #FF1E1E（クリムゾンレッド） |
| サブカラー | rgba(255,30,30,0.15)（暗めの赤グロー） |
| 英字フォント | Bebas Neue + Anton（太さの組み合わせ） |
| 和文フォント | Noto Sans JP |
| テクスチャ | ノイズ・スキャンライン・グリッチ |
| エフェクト | スクロールトリガーのフェードイン、赤い光線 |

## Blue版との差別化（完全に変える）
- Blue：サイバー・クール → v2：熱量・映画的
- Blue：整然3列グリッド → v2：非対称・ドラマティックレイアウト
- Blue：ホバーエフェクト主体 → v2：スクロール連動の大胆な登場
- Blue：情報密度高め → v2：余白と一言で刺す

## AIが作ったと感じさせないルール
1. 「カード3枚を整列」セクションを一切作らない
2. 数字に感情の注釈をつける（"8回目の大会まで、止まらなかった"）
3. セクション間に「一言だけ」の息継ぎを入れる
4. コピーに手触りを持たせる（テンプレ的な文言は全部書き直す）
5. ゲームタイトルに固有の文脈を持たせる
6. 非対称レイアウト：左右の重心を意図的にずらす

### コミュステとは
- すべてのプレイヤーに舞台を提供するDiscordコミュニティ
- ゲームジャンル横断（APEX、SF6、VALORANT、Slay the Spire 2 etc.）

### プラットフォーム構成
- Discord：日常交流・大会運営・メンバーのつながり

## 各ページの核心コピー方針（SCRIBEへ）

### index.html
- キャッチ：「眠れる才能を、主人公へ。」（継続）
- サブ：NO SKILL / NO NAME / NO EXPERIENCE → それでいい。ここにいろ。
- コミュステへの導線を最初のCTAに

### community.html
- キャッチ：「舞台は、ここにある。」
- コミュステ = プレイヤーが「主人公になれる場所」の証明
- 数字を物語で語る

### ambassador.html
- キャッチ：「コミュニティの顔に、なれ。」
- スキル不問。でも本気の人間だけが変える
- 応募への強いCTA
